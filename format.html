Json (JavaScript Object Notation)

Müxtəlif sistemlər arasında çox sayda məlumat göndərilərkən xüsusi bir formatda göndərilməyə ehtiyac duyulur. Bu format elə olmalıdır ki, məlumatlar rahat şəkildə oxuna və istifadə oluna bilsin. Bunun üçün mövcud olan formatlardan biri JSON formatıdır. Json formatının yazılışı aşağıdakı şəkildədir: 

{"ad":"Orxan", "soyad":"Abdullayev"}

müxtəlif məlumatlar bir-birindən vergüllə ayrılır. İlk olaraq dırnaq arasında məlumatın adı, sonra isə “:” simvolu qoyularaq məlumatın özü yazılır. Yazılar dırnaq arasında yazılsa da ədədləri dırnaq arasında yazmasaq da olar. 

{"ad":"Orxan", "soyad":"Abdullayev", "yas":39}

JSON Javascript obyektlərini mətn kimi saxlamağa imkan verir. Json mətni asanlıqla Javascript obyektinə dönüşdürülərək obyekt şəklində istifadə oluna bilər. Bunun üçün JSON.parse() funksiyasından istifadə edə bilərik. 

            var x=`{"ad":"Orxan", "soyad":"Abdullayev", "yas":39}`;
var y=JSON.parse(x);
console.log(y);
document.write(y.ad);

Beləliklə, dırnaqlar arasında olan json mətnini Javascript obyektinə çevirib istifadə edə bilirik. İlk olaraq bütün obyekti konsol üzərində göstəririk. Konsolda bütün obyektin göstərildiyini görə bilərsiniz. Sonra isə obyektin ad elementini brauzerdə çap edirik.

Yuxarıda yazdığımız kodun əksini də yaza bilərik. Yəni, Javascript obyektini Json yazılışına çevirməyimiz mümkündür. Bunun üçün Json.stringify() funksiyasından istifadə edə bilərik.
Nəticədə obyektin Json formatına çevrilərək mətn kimi yazıldığını görəcəyik.

Əlavə olaraq qeyd edək ki, Json proqramlaşdırma dilindən asılı deyil və Jsonla göndərilmiş məlumatlar bütün proqramlaşdırma dilləri tərəfindən istifadə oluna bilər. Json ilə məlumatlar obyekt içində obyekt şəklində də saxlana bilər.
Burada artıq bir Json məlumatının içərisində bir neçə şəxsə aid məlumatı saxlaya bilirik. Burada g.person1.name yazaraq birinci şəxsin adını götürmüş oluruq. Əgər, ikinci şəxsin soyadini götürmək istəsək g.person2.soyad yazmalyıq. 

//Json ilə məlumatlar obyekt içində obyekt şəklində də saxlana bilər.
            var h = `{
              "person1":{"name":"Aynure","soyad":"Aliyev"},
              "person2":{"name":"Resmiyye","soyad":"Merdanli"}
              }`;
            var g = JSON.parse(h);
            //document.write(g.person1.name);
            //document.write(g.person2.soyad);
            document.getElementById("ad").innerHTML = g.person1.name;
            document.getElementById("soyad").innerHTML = g.person2.soyad;

Json daxilində birbaşa çoxluq da yaza bilərik

Burada çoxluq (array) şəklində yazılmış Json mətnini Javascript çoxluğuna çevirib istifadə edə bilirik. 

                    //Json daxilində birbaşa çoxluq da yaza bilərik.
                    var n = `["Cemile","Aynure","Ceyran"]`;
                    var m = JSON.parse(n);
                    //document.write(m[0]);
                    document.getElementById("coxluq").innerHTML = m[0];
